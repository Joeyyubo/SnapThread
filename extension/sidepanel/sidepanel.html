How it works

Lives in the browser side panel with the page. Use it on any HTTP/HTTPS site you are reviewing—not only localhost. Capture a single element or drag a rectangle across multiple components, add a note, then post or copy Markdown. Drag the splitter if the page feels tight. Pin this extension in the toolbar to reopen the panel anytime.

Cross-origin iframes cannot be picked from the top page — open that URL in its own tab. Paste a GitHub pull or issue link (same comment API).

Enterprise examples: …/pull/1 or …/issues/1

Use the web page on the left, not this panel. Element: hover for a blue outline and label. Region: click-drag a dashed rectangle across any area. While picking: Esc on the page or in this panel, or Cancel capture below.
